# AstrBot E2B 云沙箱插件

使用 E2B 云沙箱安全执行 Python 代码，支持通过 LLM 自然语言调用。

## 功能特性

- 🔒 安全的云端沙箱环境执行代码
- 🤖 LLM 自动识别并调用代码执行
- ⚡ 异步执行，不阻塞机器人
- 📝 完整的错误信息和输出捕获

## 安装

1. 在 AstrBot 管理面板中搜索并安装此插件
2. 或使用仓库地址安装：`https://github.com/sl251/astrbot_plugin_e2b_sandbox`

## 配置

1. 前往 [E2B](https://e2b.dev) 注册账号并获取 API Key
2. 在 AstrBot 插件配置页面填入 API Key

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| e2b_api_key | E2B API 密钥 | 空 |
| timeout | 执行超时时间（秒） | 30 |
| max_output_length | 最大输出长度 | 2000 |

## 使用示例

直接用自然语言与机器人对话：

- "帮我计算 1 到 100 的质数之和"
- "写一段代码生成斐波那契数列前 20 项"
- "执行这段 Python 代码: print('Hello World')"

机器人会自动识别需要执行代码的场景，调用云沙箱执行并返回结果。

## 注意事项

- E2B 免费额度有限，请合理使用
- 代码在隔离的沙箱环境中执行，无法访问本地文件
- 支持大部分 Python 标准库和常用第三方库
